Pangea Ports

A single Pages landing site for the imported Android/WASM Pangea ports. Use the direct play links for browser testing, and use the level-entry examples where the underlying game already supports menu-skipping or direct level boot.

Game Play Direct-level example Notes
Billy Frontier Launch — No standardized skip-to-level entrypoint is currently documented in this port.

Nanosaur Launch ?level=0&skipMenu=1 Best current direct-boot example for level-editor workflows.

Otto Matic Launch ?level=1 Includes terrain override hooks for editor-driven testing.
