Billy Frontier

Direct-launch logic now accepts the same query-string style used elsewhere in the monorepo, while preserving legacy hash support.

Supported: ?level=N and ?terrainFile=...
Launch gameDocs

Direct level boot

Jump directly into a specific duel, shootout, stampede, or target-practice area.

./BillyFrontier-Android/game/billyfrontier.html?level=1

Native equivalent: --level 1

Level data override

Upload a custom terrain file into MEMFS or point the build at an existing colon-style terrain path.

Module.FS.writeFile('Data/Terrain/custom_level.ter', bytes)
Module.ccall('BF_SetTerrainFile', null, ['string'], [':Terrain:custom_level.ter'])
