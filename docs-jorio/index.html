Pangea Ports — Alternative WASM Builds

All 8 games built from jorio's upstream source using Emscripten's LEGACY_GL_EMULATION fixed-function translation layer.

⚙️ Renderer approach

Instead of rewriting the games' OpenGL rendering code to use GLES2, these builds use Emscripten's built-in fixed-function translation layer (-sLEGACY_GL_EMULATION=1). This is the same concept as LostMyCode/d3d9-webgl — a fixed-function graphics API translated automatically to WebGL shaders — but for OpenGL rather than Direct3D 9.
